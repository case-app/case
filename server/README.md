# ABACUS BACK

## Description

"Abacus back" is the server-side application for "Abacus front".

## Installation

- Create a new mysql database
- Copy `.env.example` into `.env`

```bash
$ npm install

```

Seed dummy data and files

```bash
$ npm run seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
